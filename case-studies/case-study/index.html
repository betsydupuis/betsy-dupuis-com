The Challenge

When joining P2 Energy Solutions, I was tasked with leading a redesign and code refactoring of P2 Tobin Insight, a user interface for querying oil and gas data to build custom reports.

User Interface Engineering

The application requirements were for the application to be rebuilt from the ground up in AngularJS 1. The team had no experience with AngularJS (aside from myself) or preexisting coding guidelines or component library.

User Experience Design

The application needed to be redesigned to usability and aesthetics while taking care to not remove any existing functionality. The design also needed to take into considerations, technical limitations that would prevent certain UI improvements.

The Process

Research and documentation

Early on I identified that structure and documentation would be crucial to the implementation of this project. The early stages of the of the project were spent defining users, use cases, and existing features.

Whiteboard wireframing

At the beginning of each iteration, I would take my findings – user experience problems in existing app and present them to the team as a whiteboard drawing from sketches I had previously made.

Prototype coding & design

After a wireframe design is approved, I’ll start using HTML, CSS, and AngularJS to build a functional prototype of the features planned.

Gathering feedback

At the end of each iteration, I will present the most recent design and gather feedback for the next iteration.
